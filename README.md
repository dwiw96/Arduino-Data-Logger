# Arduino-Data-Logger
record and save data from 10 sensor to micro sd card

## Component
* Arduino Atmega 2560
* DHT11
* DHT22
* MAX6675
* BH175FVI-E
* Micro SD Card
* Modul Micro SD Card Reader
